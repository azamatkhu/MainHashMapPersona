# Ejercicio MainHashMapPersona

**Crea un HashMap donde la clave sea el nombre de una persona y el valor su
edad. Añade varios elementos y muestra la edad de una persona concreta.**

En este ejercicio, voy a crear un hashmap Map<String, Integer>, donde la clave es un nombre (String) y valor es edad (int).
He añadido varios elementos y he mostrado la edad de una persona concreta.
